# Forecast & Chill
Ett grupparbete inom Javascript för kursen HTML, CSS och Javascript.

Sidan hämtar en användares geolocationdata om den är tillgänglig, alternativt att man skriver in en plats manuellt i sökfältet, och hämtar sedan väderdata utifrån det och visar upp antingen en aktivitet eller filmer beroende på om det är bra väder eller inte.

Syftet med uppgiften var att lära sig hur man skaffar och använder en API-nyckel, planerar programstruktur och tolkar dokumentation för nya funktioner som geolocation.

Vi skulle också öva på att samarbeta i grupp med agila metoder, läsa och ge feedback på andras kod, och dela upp stora uppgifter i mindre delar.

## Lärdomar

Vi har lärt oss att koda i javascript och att använda API:er och vår lärdom kopplat till det är att våga leta efter andra API:er, gå utanför instruktioner och kolla med läraren. 

T.ex. som att använda andra API:er än de som blev föreslagna, i vårt fall TMDB, open meteo, openstreetmap och countriesnows API för stadsdata

Vi hade en avsikt att förhålla oss till ett agilt arbetssätt, men det fallerade då vår gruppdynamik tog över och vi samarbetade på de flesta av uppgifterna.
Det har sin plats men är inte alltid rätt alternativ.
